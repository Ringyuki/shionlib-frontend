## Introduction

The Kun Galgame Patch is an open-source, free, zero-barrier, hand-coded, state-of-the-art download site for Galgame patches.

Here is the GitHub repository—feel free to drop us a star:

[https://github.com/KUN1007/kun-galgame-patch-next](https://github.com/KUN1007/kun-galgame-patch-next)

## Tech Stack

The Kun Galgame Patch is built with:

Core: Next.js + Prisma + PostgreSQL + PM2 + Cloudflare Tunnel + S3 Object Storage

Details

> Next.js 15, React 19, Node.js, AWS S3 API, S3 Object Storage, Microsoft Graph, HeroUI, Prisma ORM, Argon2, BLAKE3, Framer Motion, Redis, Sharp, Unified.js, Milkdown, Zod, Zustand, PM2, Cloudflare R2, Cloudflare Worker, Cloudflare CDN, Cloudflare Tunnel
