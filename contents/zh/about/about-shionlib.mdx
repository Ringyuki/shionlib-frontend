## 介绍

👋 嗨，欢迎来到 Shionlib!

如你所见，这是一个视觉小说/Galgame资源站，我们提供包括 游戏本体、翻译补丁 在内的资源下载服务，免费，零门槛，且不会限速。

Shionlib 是**免费**且**开源**的，你可以在 [我们的 GitHub 仓库](https://github.com/Ringyuki/shionlib-frontend) 上找到我们的源代码。如果你有任何问题或建议，欢迎提交 [Issue](https://github.com/Ringyuki/shionlib-frontend/issues) 或 [Pull Request](https://github.com/Ringyuki/shionlib-frontend/pulls)。

## 原则与网站定义

Shionlib 始终是**免费**且**开源**的，我们不会向用户收取任何费用。Shionlib 提供的所有资源均由用户上传，我们无法保证资源的完整和安全性。如果你想上传游戏资源，请阅读指南中的 [上传游戏文件](/zh/docs/guides/upload-game-files)。

如果你所在的国家/地区有合规的购买途径，你可以在游戏条目的**相关链接**处找到游戏的商店或者官方网站链接，如果有所缺失，你也可以通过**编辑**功能添加缺失的相关链接。

正如前文所提到的，在大部分国家和地区，我们往往缺少合规的正版视觉小说/Galgame购买途径。此外，有许多优秀的开发商已经因为各种原因消失在历史中，甚至在整个互联网中都难以找到他们遗留的作品。

因此，我们构建 Shionlib 的初衷之一，是为全年代供视觉小说/Galgame建立一个完整、可控的归档，同时让更多人能够有机会接触到这**绝妙的艺术体裁**。

如果你有能力，我们**非常推荐**你购买正版游戏，支持你喜欢的作品和开发商，让我们热爱的文化能够延续下去。
